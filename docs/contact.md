# RuleDSL — Contact & Evaluation

## Interested in RuleDSL?

We are looking for pilot partners in fintech, compliance, and risk management.

If your team is evaluating embedded rule engines for production use, reach out — we will set up a structured evaluation at no cost.

## How It Works

1. **Reach out** — contact us via the links below
2. **Receive the SDK** — full delivery packet with headers, binaries, bindings, and documentation
3. **Integrate and test** — follow the [evaluation playbook](evaluation_playbook.md) with your own data
4. **Validate** — confirm deterministic, reproducible results across your target environments
5. **Discuss next steps** — if it fits, we tailor the engagement to your deployment scope

## What You Get

| Item | Included |
|------|----------|
| SDK delivery | Full versioned artifact packet (headers + binaries + docs) |
| Language bindings | Python and C# wrappers |
| Examples | Working integration examples with expected outputs |
| Documentation | Complete technical documentation set |
| Support | Email-based technical support during evaluation |
